Dynamic Memory Allocator

My sophomore year, I made as part of my systems programming class a dynamic memory allocator similar to that used by the C standard library. In fact, it was written using the same function names as malloc, calloc, realloc, and free. Since this was a class assignment, it was relatively limited in scope on topics like thread-safe functionality. However, the program could be used for any single-threaded Unix system function. The repository that stores my code is private, but access is available upon request.


Concurrent Multiplayer Game

My freshman year, as part of my second course in introductory computer science, a partner and I used the Phaser.io game library, the Socket.io API, and a custom Node.JS server to make a concurrent multiplayer game. While the game itself was arbitrary to the assignment, I learned to work effectively as part of a team, and I was introduced to the concepts of sockets and communication between API's.
